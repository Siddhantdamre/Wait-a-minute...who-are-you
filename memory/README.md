# memory

Long-term, autobiographical, and schema-based memory components for the broader project.

## What Lives Here
- `autobiographical.py` and `autobiographical_store.py`: autobiographical memory logic.
- `long_term.py`: long-term storage abstractions.
- `schemas.py`: memory schema definitions.

## Notes
- This folder supports the broader cognitive system and can complement DEIC’s own memory layer.
