# core

General-purpose cognitive and orchestration modules from the broader project outside the DEIC-specific package.

## What Lives Here
- `interaction.py`, `learning.py`, `reasoning.py`: broader cognitive workflow pieces.
- `narrative.py` and `narrative_state.py`: narrative/state handling.
- `verification.py`: verification utilities.

## Notes
- This folder is broader and older than `deic_core/`.
- For the frozen bounded cognitive subsystem, use `deic_core/` first.
