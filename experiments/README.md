# experiments

One-off studies, validation harnesses, transfer runs, and historical analysis scripts.

## What Lives Here
- `cyber_transfer/`: cyber diagnosis transfer environment and adapter.
- `clinical_transfer/`: clinical transfer environment and adapter.
- top-level validation and trace scripts such as planner, anomaly, and structure checks.
- `archive/`: preserved negative-result or support scripts that are no longer on the default path.

## Notes
- This folder is intentionally exploratory.
- Benchmark-grade packaged runs live under `benchmarks/`.
