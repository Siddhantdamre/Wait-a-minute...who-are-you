## Technical Verdict

**(a) Does memory help in stationary settings?**
  YES — average accuracy lift of +7.8% across domains.

**(b) Does memory hurt under distribution shift?**
  NO — average delta of +4.4%; memory does not measurably hurt.

**(c) Recommendation: default-off, optional, or standard?**
  OPTIONAL-ON — memory provides benefit and does not hurt under shift. Ship as opt-in default-on.
