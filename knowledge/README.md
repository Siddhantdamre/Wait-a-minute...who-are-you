# knowledge

Knowledge representation and retrieval helpers used by the broader system.

## What Lives Here
- `rag.py` and `rag.txt`: retrieval-augmented workflow notes/helpers.
- `values.py` and `value_graph.py`: value/knowledge graph utilities.
- `vectorstore.py`: vector-backed storage/retrieval support.

## Notes
- This folder supports knowledge access rather than DEIC’s core hidden-state inference loop.
