# expression

Output and presentation modules for voice, narration, emotional rendering, and storyboard-style expression.

## What Lives Here
- `voice.py`, `narration.py`: spoken or narrated output helpers.
- `emotion.py`: affect/expression logic.
- `storyboard.py`, `orchestrator.py`: higher-level presentation flow.

## Notes
- This is presentation-oriented support code, not part of the frozen DEIC benchmark core.
