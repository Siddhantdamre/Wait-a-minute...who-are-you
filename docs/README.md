# docs

Project documentation: milestone notes, release bundles, design notes, logs, and benchmark writeups.

## What Lives Here
- `milestones/`: frozen milestone documents.
- `releases/`: submission and release-facing artifacts.
- `designs/`: design-only notes for future work.
- `logs/`: historical closeout and experiment notes.
