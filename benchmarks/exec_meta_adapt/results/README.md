# Benchmark Results

This source-controlled directory is retained only as a breadcrumb from the initial scaffold.

- `results/deic_cogbench/package_run.json`
- `results/deic_cogbench/package_report.md`
- `results/deic_cogbench/ablations_train.json`

This keeps generated benchmark artifacts outside committed source paths.
