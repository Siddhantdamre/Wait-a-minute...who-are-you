# AGI_Byzantine_Benchmark

Legacy benchmark components for Byzantine-style trust, coordination, and hidden-state simulation work.

## What Lives Here
- `agent_schema.py`: agent and belief-schema definitions.
- `benchmark_task.py` and `environment.py`: task/environment setup.
- `metrics.py`: scoring helpers.
- `run_simulation.py`: runnable simulation entrypoint.

## Notes
- Treat this folder as older benchmark groundwork rather than the main DEIC-CogBench package.
- For the current benchmark packaging work, start in `benchmarks/`.
